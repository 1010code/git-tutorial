# CLONING & BRANCHING

## clone repository

```
git clone url
```

## 開啟分支
```
git branch cat
```
檢查目前所有分支
```
git branch
```
## 進入指定分支
```
git checkout cat
```
commit 後回主分支
```
git checkout master
```
合併分支
```
git merge cat
```
刪除cat分支
```
git branch -d cat
```

## 建立分支並進入
```
git checkout -b admin
```
